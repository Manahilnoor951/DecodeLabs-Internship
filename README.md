# 📋 Overview

**Cyber Security Projects Portfolio** is a collection of three practical cybersecurity projects completed during the **DecodeLabs Industrial Training Program (Batch 2026)**.

These projects introduce essential cybersecurity concepts including password security, classical encryption, and phishing detection through hands-on implementation and security analysis.

---

# ✨ Projects

## 🔐 Project 1 — Password Strength Checker

### Features

* 🔑 Analyze password strength
* 📏 Validate minimum password length
* 🔠 Detect uppercase letters
* 🔢 Detect numeric digits
* ✳️ Detect special characters
* 📊 Classify passwords as Weak, Medium, or Strong
* 🛡️ Encourage secure password creation

---

## 🔒 Project 2 — Caesar Cipher Encryption & Decryption

### Features

* 🔐 Encrypt user messages
* 🔓 Decrypt encrypted messages
* 🔑 User-defined shift key
* 🔠 Supports uppercase & lowercase letters
* ✍️ Preserves spaces and symbols
* 📖 Demonstrates basic cryptography concepts

---

## 🛡️ Project 3 — Phishing Awareness Analysis

### Features

* 📧 Analyze phishing emails
* 🔍 Detect suspicious URLs
* 🚩 Identify phishing keywords
* ⚠️ Recognize phishing red flags
* 📑 Explain security risks
* 🧠 Improve cybersecurity awareness
